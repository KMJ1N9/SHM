## 建议测试流程（审核人员操作指南）

### 基础体验（5-8 分钟）

1. **登录** → 使用账号 1 手机号 + 微信授权登录
2. **浏览** → 首页浏览商品 → 展开筛选栏 → 按分类筛选
3. **搜索** → 搜索"教材" → 切换排序方式
4. **详情** → 点击任意商品 → 查看图片轮播/描述/卖家信息
5. **聊天** → 点击"聊一聊" → 发送消息
6. **下单** → 返回商品详情 → 点击"我想要" → 创建订单
7. **订单** → 切换到"订单"Tab → 查看买卖订单列表
8. **通知** → 切换到"消息"Tab → 查看系统通知/未读数量
9. **个人** → 切换到"我的"Tab → 查看信誉分/设置/用户协议

### 管理功能（可选，2-3 分钟）

1. 切换到账号 3（管理员）登录
2. 管理员首页 → 查看数据看板
3. 用户管理 → 搜索用户 → 查看详情
4. 工单处理 → 查看举报列表 → 受理工单

---

## 注意事项

- 上述测试账号为**后端 mock 模式**下的虚拟账号。实际审核时，若已对接真实微信登录，需使用审核人员自己的微信账号登录后，在管理后台手动设置角色/信誉分。
- 聊天功能依赖腾讯云 IM SDK，需在微信开发者工具中开启"不校验合法域名"选项或配置 IM WebSocket 域名白名单。
- 图片上传功能依赖腾讯云 COS，同样需要配置域名白名单。

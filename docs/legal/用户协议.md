## 二、用户注册与账号管理

### 2.1 注册方式

本平台使用微信小程序 `getPhoneNumber` 能力获取您的微信绑定手机号，作为唯一身份标识完成注册与登录。首次授权即自动注册，后续授权即登录。您无需设置额外密码。

2.2.1 您的账号与微信账号绑定，请妥善保管您的微信账号安全。因微信账号被盗用、借用导致的一切后果，由您自行承担。

2.2.3 本平台严禁账号买卖、出借、出租。一经发现，本平台有权永久封禁相关账号。

### 2.3 注册资格

本平台目前仅面向广州应用科技学院肇庆校区在校学生开放。您应如实填写班级、宿舍楼栋等信息。本平台保留在后续迭代中接入学校统一身份认证平台的权利。

## 三、服务内容

3.1 本平台为 C2C 二手交易信息撮合平台，提供以下服务：

| 服务 | 说明 |
|------|------|
| 商品发布 | 用户可发布闲置物品信息（含图片、描述、价格等） |
| 商品浏览与搜索 | 瀑布流展示、分类筛选、关键词搜索 |
| 站内即时通讯（IM） | 买卖双方在平台内沟通协商 |
| 订单管理 | 交易状态跟踪、确认收货 |
| 双向评价 | 交易完成后双方互评 |
| 举报与客服 | 违规行为举报、工单处理 |

3.2 **特别提示**：MVP 阶段，本平台不介入资金流转。交易款项由买卖双方自行在线下结算。本平台仅记录交易状态，不为任何一方提供支付担保或资金托管服务。

3.3 本平台保留根据运营需要增加、减少或调整服务内容的权利。

## 四、用户行为规范

### 4.1 禁止发布的商品

以下商品严禁在本平台发布：

| 类别 | 示例 |
|------|------|
| 违法违规物品 | 毒品、管制刀具、枪支弹药、易燃易爆品 |
| 侵害他人权益 | 盗版、侵权复制品、假冒伪劣商品 |
| 人身安全风险 | 已拆封的食品/药品/化妆品、过期商品 |
| 校园违禁品 | 大功率电器（校规禁止的）、考试作弊器材 |
| 不适宜物品 | 烟草、电子烟、酒类、活体动物、虚拟账号/代练服务 |
| 其他 | 法律法规禁止交易或本平台认为不适宜交易的物品 |

### 4.2 用户行为禁止

在使用本平台过程中，您不得从事以下行为：

- 发布虚假、误导性商品信息
- 恶意下单或虚假确认收货（刷单/刷评价）
- 在站内 IM 中辱骂、骚扰、威胁其他用户
- 利用平台实施诈骗、传销、非法集资等违法犯罪行为
- 诱导其他用户脱离平台交易（私下交换微信、QQ 等联系方式进行交易）
- 使用爬虫、脚本等自动化工具采集平台数据
- 利用平台发布广告、商业推广信息
- 其他违反法律法规、校规校纪或公序良俗的行为

## 五、交易规则

### 5.1 交易流程

```
卖家发布商品 → 买家浏览/搜索 → 站内IM议价 →
线下约定面交地点 → 当面验货/一手交钱一手交货 →
确认收货 → 双向评价
```

### 5.2 交易须知

5.2.1 交易价格由买卖双方自行协商确定，本平台不参与定价。

5.2.2 面交地点建议选择校园内人流量较大的公共区域（如食堂门口、图书馆门前、宿舍楼大堂等），注意人身安全。

5.2.3 面交前请仔细核验商品成色、功能是否与描述一致。确认收货后视为您已认可商品状态，后续纠纷本平台仅提供工单处理协助，不承担直接赔偿责任。

5.2.4 买卖双方任一方点击"已面交"后，另一方应在 **3 天内**完成确认收货。逾期未确认的，系统将自动确认收货并触发双向评价。

5.2.5 订单创建后 **7 天内**未进入"已面交"状态，系统将自动取消订单。

### 5.3 线下交易免责

**本平台强烈建议用户通过站内 IM 沟通并在平台内完成交易全流程。** 若用户通过站内 IM 交换微信、QQ 等第三方联系方式，脱离本平台进行私下交易，发生任何纠纷或损失，本平台不承担责任。详细免责条款见《[免责声明](免责声明.md) §三 私下交易免责》。

## 六、评价与信誉体系

### 6.1 评价规则

- 交易完成后双方需在 3 天内完成双向评价
- 评价内容应真实、客观，不得恶意差评
- 所有评价均实名可见，不可匿名

### 6.2 信誉分体系

| 项目 | 分值 |
|------|:--:|
| 新用户初始分 | 100 |
| 信誉分上限 | 200 |
| 举报成立（被举报方） | -30 |
| 完成交易（卖家） | +2 |
| 收到好评 | +1 |
| 收到差评（2 星及以下） | -5 |

| 信誉分区间 | 限制 |
|-----------|------|
| ≥60 | 全部功能正常使用 |
| 30-59 | 可浏览/下单，不可发布商品 |
| < 30 | 仅可浏览商品，不可发布/下单 |
| ≤0 | 标记为高风险用户，限制登录并人工回访 |

信誉分低于 60 分的用户，可通过积极完成交易逐步恢复分数。

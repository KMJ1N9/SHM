## 引言

**校园二手交易小程序**（以下简称"本平台"或"我们"）是面向广州应用科技学院肇庆校区在校学生的校内 C2C 二手交易平台。我们深知个人信息保护的重要性。本《隐私政策》依据《中华人民共和国个人信息保护法》（PIPL）、《网络安全法》、《数据安全法》及相关法律法规制定，旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

**个人信息处理者：** 校园二手交易小程序开发团队（以下简称"我们"）。联系方式见[§九 联系我们](#九-联系我们)。

**请您务必仔细阅读并充分理解本隐私政策的全部内容。** 您通过微信授权登录本平台，即视为您已同意本隐私政策。如您不同意本隐私政策的任何条款，请立即停止使用本平台。

## 一、我们收集的信息

### 1.1 必要个人信息（不提供则无法使用核心服务）

| 信息类型 | 字段 | 收集方式 | 目的 |
|---------|------|---------|------|
| **手机号码** | 微信绑定手机号 | 微信 `getPhoneNumber` 授权 | 用户身份标识、注册/登录、客服联系 |
| **微信头像与昵称** | 用户自行设置 | 用户自主编辑（默认值："微信用户"） | 平台内展示、其他用户识别 |

### 1.2 用户自主提供的信息

| 信息类型 | 字段 | 目的 | 是否必填 |
|---------|------|------|:--:|
| 个人资料 | 班级、宿舍楼栋、年级 | 增强交易信任（校园场景） | ❌ 选填 |
| 商品信息 | 图片、名称、描述、价格、成色、交易地点 | 商品发布与展示 | ✅ 发布商品时必填 |
| 聊天内容 | 站内 IM 文本、图片消息 | 交易沟通 | ❌ 使用 IM 功能时产生 |
| 评价内容 | 评价文字、评分 | 双向评价体系 | ✅ 评价时必填 |
| 举报内容 | 举报描述、截图证据 | 违规处理 | ✅ 举报时必填 |

### 1.3 自动收集的技术信息

| 信息类型 | 字段 | 目的 |
|---------|------|------|
| 设备信息 | 设备型号、操作系统版本、微信版本 | 兼容性适配、问题排查 |
| 日志信息 | 访问时间、页面路径、接口调用记录、错误日志 | 系统稳定性维护、安全审计 |

> 注：本平台**不会**请求您的精确 GPS 定位权限。

## 二、我们如何使用信息

2.1 您的个人信息仅用于以下目的：

| 用途 | 说明 |
|------|------|
| 账号服务 | 注册/登录、账号安全保护 |
| 商品展示 | 在商品详情页、瀑布流展示商品信息及卖家头像/昵称 |

| 服务优化 | 统计用户使用行为，改进产品 |

2.2 我们不会将您的个人信息用于个性化推荐广告或出售给第三方。

2.3 如果您拒绝提供必要个人信息，您将无法完成注册和使用本平台的核心功能（浏览商品、发布商品、IM 聊天、交易）。

## 四、信息存储与安全

### 4.1 存储位置

您的个人信息存储于**中华人民共和国境内**的腾讯云服务器（广州区域）。

### 4.2 存储期限

| 信息类型 | 存储期限 |
|---------|---------|
| 账号信息（手机号、头像、昵称等） | 账号存续期间 + 注销后 **30 天**内删除 |
| 商品信息 | 商品发布至下架/删除期间，删除后 7 天内清理 |
| IM 聊天记录 | 腾讯云 IM 云端保留 **7 天**，本地缓存由微信管理 |
| 交易记录 | 交易完成后保留 **5 年**（财务记录要求） |
| 举报/工单记录 | 保留 **2 年**（合规审计要求） |
| 日志信息 | 保留 **90 天** |

> **MVP 阶段说明：** 上述保留期限为本平台的隐私承诺。当前 MVP 阶段，日志轮转由 winston-daily-rotate-file（`maxFiles: '90d'`）自动执行；举报/工单超期清理由月度 cron 软删除；交易记录因数据量极低暂不设置自动清理，将留存至服务器存储上限后手动处理。具体实现详见《[技术架构文档 §11.4](../技术架构文档.md)》。

### 4.3 安全措施

| 措施 | 说明 |
|------|------|
| 传输加密 | 全站 HTTPS（TLS 1.2+），API 通信加密 |
| 访问控制 | JWT 双 Token 认证 + 角色权限矩阵 |
| 数据隔离 | 数据库文件权限控制，仅应用进程可读写 |
| 敏感词过滤 | 站内 IM 消息实时敏感词检测 |
| 安全审计 | `admin_logs` 表记录所有管理员敏感操作 |
| 最小权限 | 腾讯云 COS STS 临时凭证，最小化权限范围 |

本平台由校园二手交易小程序开发团队维护。我们会尽力保护您的个人信息安全，但受限于资源和规模，无法提供企业级安全保障。如发生数据安全事件（如数据泄露），我们将立即采取补救措施，并依法向您和有关监管部门报告。

## 五、您的权利

根据《个人信息保护法》，您享有以下权利：

| 权利 | 说明 | 行使方式 |
|------|------|---------|
| **查阅权** | 查看我们收集的您的个人信息 | 小程序"我的"页面可查看个人资料；完整数据导出请联系客服 |
| **更正权** | 修改不准确或不完整的个人信息 | "我的"→"编辑资料"自主修改 |
| **删除权** | 在以下情形要求删除个人信息：(1) 处理目的已实现或无法实现；(2) 我们停止提供服务；(3) 您撤回同意；(4) 我们违法处理您的信息 | 联系客服提交删除申请 |
| **撤回同意** | 撤回对收集和使用个人信息的同意 | 小程序设置中撤回授权；撤回后不影响撤回前已基于同意进行的处理 |
| **注销账号** | 注销账号并删除个人信息 | "我的"→"关于平台"→"注销账号"；注销后 30 天内完成数据清理 |
| **数据可携** | 将个人信息转移至其他平台 | 联系客服，我们将按国家标准格式提供可机读的数据导出 |

我们将在收到您行使权利的请求后 **15 个工作日内**做出答复。

## 六、未成年人保护

6.1 本平台面向高等院校在校学生（一般为年满 18 周岁的成年人）。

6.2 若您是未满 14 周岁的未成年人，请勿使用本平台。若您是已满 14 周岁但未满 18 周岁的未成年人，请在监护人同意和指导下使用本平台。

6.3 如我们发现误收集了未满 14 周岁未成年人的个人信息，将在 7 天内予以删除。

## 七、Cookie 及同类技术

# UI设计系统文档：校园二手交易小程序

## 一、设计原则

### 1.1 设计理念
- **简约至上**：以灰色为主色调，减少视觉干扰，让用户专注于内容本身
- **信息高效**：紧凑布局，最大化信息密度，便于快速浏览
- **交互友好**：舒缓的过渡动画，清晰的操作反馈

### 1.2 核心价值观
- **清晰可读**：良好的对比度和层级关系
- **一致性**：统一的设计语言贯穿全局
- **易用性**：直观的交互模式，降低学习成本

## 五、组件库

### 5.1 按钮

#### 5.1.1 按钮类型

| 类型 | 样式 | 用途 |
|------|------|------|
| 主按钮 | 蓝色渐变背景，白色文字 | 主要操作、确认按钮 |
| 次按钮 | 白色背景，蓝色边框和文字 | 次要操作、取消按钮 |
| 文字按钮 | 无背景，蓝色文字 | 辅助操作、链接 |
| 禁用按钮 | 灰色背景，浅灰文字 | 不可操作状态 |

#### 5.1.2 按钮状态

| 状态 | 效果 |
|------|------|
| 默认 | 标准样式 |
| 悬停 | 轻微放大(1.02x)、颜色加深、阴影增强 |
| 点击 | 轻微缩小(0.98x)、阴影内收 |
| 禁用 | 降低透明度(60%)、去除交互效果 |

#### 5.1.3 按钮尺寸

| 尺寸 | 高度 | 字体大小 |
|------|------|---------|
| 大 | 48px | 16px |
| 中 | 40px | 14px |
| 小 | 32px | 12px |

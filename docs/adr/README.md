## ADR 模板

新建 ADR 时，复制以下模板到 `docs/adr/NNNN-标题.md`：

```markdown
# ADR-NNNN：决策标题

- **日期：** YYYY-MM-DD
- **状态：** 提议 | 已采纳 | 已废弃 | 已替代
- **决策者：** [姓名/角色]

## 背景

[为什么需要做这个决策？当前面临什么问题？]

## 决策

[选择了什么方案？]

## 考虑的替代方案

| 方案 | 优点 | 缺点 | 为何未选 |
|------|------|------|---------|
| A   |      |      |         |
| B   |      |      |         |

## 影响范围

[这个决策影响哪些模块/文件/团队？]

## 后果

### 正面
[这个决策带来了什么好处？]

### 负面
[引入了什么技术债或限制？]

## 相关

- 关联 ADR: [编号]
- 关联文档: [路径]
```

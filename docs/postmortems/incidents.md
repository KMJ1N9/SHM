## 事故统计

| 等级 | 定义 | 次数 |
|:--:|------|:--:|
| P0 | 核心功能完全不可用（登录/交易/IM），≥30% 用户受影响 | 0 |
| P1 | 重要功能不可用，10-30% 用户受影响 | 0 |
| P2 | 部分功能异常，<10% 用户受影响 | 0 |
| P3 | 轻微异常，不影响核心功能 | 0 |
